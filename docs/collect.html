# 電話番号収集タスク

## 役割
添付CSVのtel列を電話番号で埋めてください。

## CSVの列
- company_name: 会社名
- industry: 業種
- address: 住所
- search_url: Google検索URL（そのまま開ける）
- map_url: GoogleマップURL
- tel: ← ここに電話番号を入力する

## 調査手順
1. search_urlを開く
2. 公式サイトの「会社概要」「お問い合わせ」ページを確認する
3. 電話番号が見つかればtel列に入力する
4. 見つからなければ空欄のまま

## ルール
- 形式は自由（072-123-4567でも0721234567でも可）
- 代表番号のみ（FAX・携帯不要）
- 見つからない場合は空欄のまま
- 推測・創作は絶対にしない

## 出力
元のCSVと同じ形式でtel列だけ埋めて返してください。列の追加・削除・並び替えは不要です。
